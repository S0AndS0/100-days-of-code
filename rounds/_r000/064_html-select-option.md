Example of using `select` and `option` HTML elements and specific attributes...


```html
<select id="cars">
  <option value="" disabled hidden selected>
    Please select one...
  </option>
  <option value="volvo">Volvo</option>
  <option value="saab">Saab</option>
</select>
```
